<문제 1>
1. purpose of this code.
selection sort를 stable sorting 으로 해주는 것

2. input and output
input은 각각 index와 숫자들이 들어있는 것. output은 index와 정렬된 숫자를 출력하는 것 이다.

3. variable i used
가장 중요하게 생각한 변수는 바로 additional memory인 add_mem이다. 여기에 다가 변수를 따로 담아서 정렬이 stable하게 만들었다.

<문제 2>
1. purpose of this code
radix sorting 을 할 때 queue를 이용해서 정렬하는 것이 아닌 counting sort를 이용해서 정렬하는 것

2. input and output
총 1000개의 0~9999까지의 숫자를 입력 받고 output으로는 이 숫자들을 정렬하는 것 이다.

3. variable i used
가장 중요하게 생각한 변수는 exp이다. 이 변수를 통해 각각 자리수의 숫자를 counting sort로 정렬하였다.
